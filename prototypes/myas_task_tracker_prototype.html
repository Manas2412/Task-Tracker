Prototype v1

Task list · Quick create · JS priority board

The mobile-first surface for OSD, JS, and division officers. Tap around — every gesture is wired.

Try these

  • Switch role at the top of the phone — task list rescopes
  • Tap the floating "+" — Quick Create slides up; toggle "Add more details"
  • Open JS priority board — all four lanes visible side-by-side, swipe horizontally between them
  • Drag a task card from any lane into any other lane to set JS priority
  • Tap filter chips — by Today, Overdue, Urgent, JS Priority

Design notes

  • Amber accent = JS Priority signal
  • Coloured dot = priority (urgent → low)
  • Pills = status, milestone, JS lane
  • Coloured avatar = division
This is a static reference design for Lovable. All data is mock. Next prototype: Task detail screen with comments, subtasks, attachments, and the Timeline File panel.
