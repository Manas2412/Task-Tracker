Prototype v3

Timeline file detail

The Secretary's-office correspondence tracker — opens when a Timeline File is tapped from the master view or from a linked task.

Try these

  • Tap In progress — TF status sheet (different list from task statuses)
  • Tap Create task from this file — pre-fills deadline, division, and back-reference
  • Tap a linked task — would navigate to its detail screen
  • Tap either source document — full-screen preview with the actual PMO letter
  • Tap the action-document upload box
  • Tap the three-dot menu top-right

Design notes

  • Indigo accent throughout — Timeline Files use the same colour as the linked-TF card on the task detail screen
  • Secretary's comments rendered as a serif quote — formal direction, signed
  • Linked tasks reuse the same card grammar as the task list, including JS Priority left-stripe
  • Action document is a placeholder until uploaded — closes the loop
  • Reference number in monospace — looks and feels like a file number
The linked-tasks panel is the panel the Secretary scans in the weekly review. Status of each task is visible without leaving this screen.
